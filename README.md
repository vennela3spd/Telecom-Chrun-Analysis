## Project Overview
Customer churn is one of the most critical challenges faced by telecom companies. This project focuses on analyzing **customer churn patterns** using **Power BI** to help businesses understand customer behavior, identify churn drivers, and make data-driven retention strategies.

The dashboard provides a **visual and interactive analysis** of customer demographics, subscription types, usage behavior, and support interactions.

---

##  Objectives
- Analyze customer churn based on subscription type, gender, and contract length  
- Identify patterns in customer support calls and usage frequency  
- Understand spending behavior across different subscription plans  
- Provide actionable insights to reduce churn

##  Dataset Description
The dataset contains telecom customer information, including:
- Customer ID  
- Gender  
- Subscription Type (Basic, Standard, Premium)  
- Contract Length (Monthly, Quarterly, Annual)  
- Total Spend  
- Usage Frequency  
- Number of Support Calls  
- Churn Status (Yes / No)  

---

##  Dashboard Features
- **KPI Cards**
  - Total Customers
  - Churned Customers
- **Customer Distribution**
  - By Subscription Type
  - By Gender
- **Churn Analysis**
  - Churn by Subscription Type
- **Usage & Spending Trends**
  - Total Spend vs Usage Frequency
- **Support Call Analysis**
  - Support Calls by Contract Length

---

##  Key Insights
- **Basic and Standard plans** have a higher customer count compared to Premium  
- **Monthly contracts** generate more support calls, indicating higher churn risk  
- Customers with **higher support interactions** show increased churn probability  
- **Usage frequency and spending** vary significantly across subscription types
